# 이탁우 Tetris

웹 브라우저에서 즐길 수 있는 클래식 테트리스 게임입니다. HTML5 Canvas와 JavaScript를 사용하여 구현되었으며, 모바일과 PC 환경 모두 지원합니다.

## ✨ 주요 기능

*   **클래식 테트리스 메커니즘**: 7가지 테트리미노 조각, 줄 삭제, 레벨 업(속도 증가) 등 기본에 충실한 게임 플레이.
*   **고스트 조각 (Ghost Piece)**: 조각이 착지할 위치를 미리 보여주는 가이드 기능 제공 (ON/OFF 가능).
*   **홀드 (Hold)**: `Shift` 또는 `C` 키로 현재 조각을 보관하고 나중에 사용할 수 있는 전략적 기능.
*   **다음 조각 미리보기 (Next)**: 다음에 나올 조각들을 미리 보여주어 전략적인 플레이 가능.
*   **점수 및 통계**: 점수, 제거한 줄 수, 경과 시간 표시.
*   **비주얼 효과**: 
    - 각 블록 색상별 입체감 있는 디자인.
    - 줄 삭제 시 애니메이션 효과.
*   **사운드 시스템**:
    - 배경 음악 (유명한 테트리스 테마 'Korobeiniki').
    - 이동, 회전, 드롭, 줄 삭제, 게임 오버 등 다양한 효과음.

## 🎮 조작법 (Controls)

### ⌨️ PC (키보드)
| 동작 | 키 |
| :--- | :--- |
| **이동** | `←`, `→` (방향키) |
| **빠른 드롭 (내리기)** | `↓` (방향키) |
| **회전** | `Q`, `W`, `↑` (방향키) |
| **하드 드롭 (즉시 착지)** | `Space`, `Enter` |
| **홀드 (저장)** | `Shift`, `C` |

### 📱 모바일 (터치)
화면 하단의 터치 버튼을 이용합니다.
*   **← / → / ↓**: 이동 및 내리기
*   **Rot**: 회전
*   **Drop**: 하드 드롭
*   **Hold**: 홀드

## 🛠️ 기술 스택

*   **HTML5 Canvas**: 게임 렌더링.
*   **Vanilla JavaScript (ES6+)**: 게임 로직 및 상태 관리 (라이브러리 의존성 없음).
*   **Web Audio API**: 배경 음악 및 효과음 합성/재생.
*   **CSS3**: 반응형 레이아웃 및 스타일링.

## 🚀 실행 방법

1.  `index.html` 파일을 웹 브라우저로 엽니다.
2.  화면을 클릭하여 게임을 시작합니다 (오디오 정책으로 인한 사용자 인터랙션 필요).
